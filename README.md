# Smart Heading Paste

一款 Obsidian 插件，在粘贴 Markdown 内容时自动根据当前文档的标题层级调整粘贴内容中的标题级别，确保结构连贯。

## 功能

- **自动检测上下文**：粘贴时自动向上查找最近的标题，作为上下文参考
- **智能层级调整**：将粘贴内容的标题整体偏移，使其首级标题对齐到 `上下文级别 + 1`
- **无上下文保护**：当粘贴位置没有上级标题时，自动将 H1 转为 H2，避免与文档标题冲突
- **代码块保护**：可选跳过代码块内的 `#` 字符（如注释），防止误调整
- **手动触发**：支持 `Ctrl+Shift+V`（或 `Cmd+Shift+V`）手动强制触发智能粘贴

## 使用示例

**场景 A：在 H1 标题下粘贴 H3 内容**

当前文档：
```markdown
# 我的项目计划（光标在此下方）
```

粘贴内容：
```markdown
### 模块设计
#### 数据库
##### 表结构
```

自动调整为：
```markdown
## 模块设计
### 数据库
#### 表结构
```

**场景 B：在无标题区域粘贴**

当前文档开头无标题，粘贴 `# 总览\n## 细节`，开启"避免文档根级别 H1"后将自动调整为：
```markdown
## 总览
### 细节
```

## 安装

### 手动安装

1. 下载最新版本的 `main.js`、`manifest.json`、`styles.css`
2. 放入你的 Vault 目录下的 `.obsidian/plugins/smart-heading-paste/`
3. 在 Obsidian 设置中启用该插件

### 社区插件市场

等待官方审核通过后即可在 Obsidian 社区插件市场中搜索 "Smart Heading Paste" 安装。

## 设置

在 Obsidian 设置 → 第三方插件 → Smart Heading Paste 中可配置：

| 设置项 | 说明 |
|--------|------|
| 避免文档根级别 H1 | 无上下文时，自动将 H1 转为 H2 |
| 显示调整提示 | 粘贴后在右下角显示调整的偏移详情 |
| 跳过代码块 | 不调整代码块内的 `#` 字符 |
| 最大标题级别 | 限制标题最高级别（默认 6） |
